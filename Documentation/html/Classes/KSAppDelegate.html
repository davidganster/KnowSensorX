Overview

Application delegate. Handles launching and stopping the KnowServer task, as well as asking for accessibility access and handling uncaught exceptions. When an applicationShouldTerminate: message is received, the delegate will tell the KSSensorController to stop recording events, reply wth NSTerminateLater, and postpone shutdown until the KSSensorController is done stopping the recording.
