Overview

The KSMenuController class encapsulates all interaction with the status bar menu, such as setting/updating the project list, current activity etc.

Note: The KSMenu is dependant on one layout constraint: The first 3 items are reserved for “Current Project/Activty”, and . Any changes after these three items do not require any changes in code, but be careful if you insert new items at the front!
