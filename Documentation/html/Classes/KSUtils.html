executeApplescriptWithName:functionName:arguments:errorDictionary:

Executes the apple script with the given scriptName, functionName and args. Any errors that occur either during execution or loading will be written to the errorDict.

+ (NSAppleEventDescriptor *)executeApplescriptWithName:(NSString *)scriptName functionName:(NSString *)functionName arguments:(NSArray *)args errorDictionary:(NSDictionary **)errorDict

Parameters

scriptName

The file name script to be executed, without the extension.

functionName

The name of the function in the script to be executed.

args

The arguments that will be passed to the executed function.

errorDict

A pointer to a dictionary that will contain any errors that occured (or be nil after the method returns).

Return Value

The NSAppleEventDescriptor obtained by executing the apple script.

Discussion

Note: This method only searches inside the application’s main bundle, so it cannot be used to execute scripts that are not included in the application.

Warning: Executing a script might take some time, so be sure to call this on a background thread.

scaleForScreenshotQuality:

Returns a float that will be used to appropriately scale images. This method takes into consideration the screen scale (otherwise, images taken on retina-devices would be four times larger).
